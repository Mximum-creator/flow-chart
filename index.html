Flowchart 2: Search Functionality
flowchart TD A[Start: User lands on Listing Page] --> B{User wants to search for Any Business, Any Category, or Any Location?} B -- Yes --> C[User enters query in Search Button] C --> D[Redirect to Search Page] D --> E{Does query match an existing Listing Page?} E -- Yes --> F[Display Search Page: Matched Listing Page card at the top
Below: Cards for nearby locations or related categories] E -- No --> G[Use APIs to generate a new Listing Page with existing UI] F --> H{User clicks on a Listing Card?} G --> H H -- Yes --> I[Redirect to corresponding Listing Page] H -- No --> J[Stay on Search Page or end] I --> K[View listings on Listing Page] B -- No --> K K --> L[End] J --> L
